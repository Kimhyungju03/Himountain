# HiMountain

## 프로젝트 소개

HiMountain은 사용자가 산 정보를 확인하고 등산 계획을 세울 수 있는 안드로이드 애플리케이션이다.

## 주요 기능

* 국내 주요 산 정보 조회
* 현재 날씨 및 3일 예보 확인
* 등산 리뷰 작성
* 평점 기능
* 오늘 갈 산 등록
* 구글 지도 연동
* 사용자 체형 및 등산 수준 기반 AI 등산 추천 기능

## 개발 환경

* Android Studio
* Java
* RecyclerView
* SharedPreferences
* OpenWeather API
* Retrofit
