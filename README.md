# 🏔️ 하이마운틴 (HiMountain)

## 프로젝트 소개

하이마운틴(HiMountain)은 사용자가 국내 유명 산의 정보를 확인하고, 날씨를 조회하며, 등산 계획을 세울 수 있도록 도와주는 안드로이드 애플리케이션입니다.

또한 사용자의 체형 및 등산 경험을 바탕으로 적합한 산을 추천하는 기능을 제공하여 초보자도 쉽게 등산을 시작할 수 있도록 설계하였습니다.

---

## 개발 목적

* 등산 초보자들이 쉽게 산 정보를 확인할 수 있도록 지원
* 실시간 날씨 정보를 통해 안전한 등산 계획 수립
* 등산 후기 및 평점 기능 제공
* 사용자 맞춤형 산 추천 기능 제공

---

## 주요 기능

### 1. 산 정보 조회

* 한라산
* 지리산
* 설악산
* 북한산

각 산의 위치, 해발고도, 난이도, 준비물, 추천 계절 등을 확인할 수 있습니다.

### 2. 날씨 정보 조회

OpenWeather API를 활용하여 다음 정보를 제공합니다.

* 현재 온도
* 체감 온도
* 습도
* 풍속
* 현재 날씨 상태
* 3일 날씨 예보

### 3. 등산 리뷰 기능

* 리뷰 작성
* 리뷰 수정
* 리뷰 삭제
* 별점 평가 기능

### 4. 오늘 갈 산 등록

사용자가 오늘 등산할 산을 등록하고 관리할 수 있습니다.

### 5. 지도 연동

Google Maps와 연동하여 산의 위치를 바로 확인할 수 있습니다.

### 6. 맞춤형 산 추천 기능

사용자의 정보를 입력받아 적절한 산을 추천합니다.

입력 정보

* 키
* 몸무게
* 등산 경험 수준

  * 초보자
  * 중급자
  * 고급자
